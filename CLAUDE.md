# CLAUDE.md

## 语言要求

- 所有回答和思考过程均使用中文。
- 代码注释、变量命名等可根据项目实际情况选择中文或英文，但解释性文字必须使用中文。

## 每日练习项目规范

### 命名规则

- 项目目录统一放在 `D:\code\daily\` 下。
- 命名格式：`YYYY_MM_DD_HH`（年_月_日_时）。
- 当天第一个项目命名为 `YYYY_MM_DD`（省略小时后缀）。

### README 要求

每个项目必须建立 `README.md` 文件，包含以下内容：

**创建时写入：**
- 日期
- 项目建立时间
- IP 地址

**项目完成后更新：**
- 项目总结（记录了今日学习/练习的内容和成果）
- 项目结构（当前项目的文件目录树）

### 任务流程

1. 提醒到达时，按命名规则创建项目目录
2. 创建 README.md，填写日期、时间、IP
3. 进行练习/开发
4. 完成后更新 README.md，添加项目总结和项目结构
5. 提交并推送到 GitHub
